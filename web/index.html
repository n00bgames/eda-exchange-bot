Server-Side Buyback Schedule

The console runs the buyback sweep unattended, so this page does not need to stay open. Scheduled and manual server-side runs use the saved values below with the exchange selected in Exchange Controls at save time; save the schedule before using Run Sweep Now. Every run probes eligibility with a read-only query first and takes a backup only when there is something to buy. All SQL is built by the console from the bundled seed plan.

Enabling requires the scheduler:server permission to be approved in the console's Addons panel. The first run fires one full interval after enabling; the schedule survives console restarts. While the server-side schedule is enabled, the in-page auto buyback checkbox is turned off: concurrent sweeps are safe at the database level but redundant, and each takes its own backup.
